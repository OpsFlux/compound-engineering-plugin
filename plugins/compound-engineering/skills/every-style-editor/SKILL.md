# 每个样式编辑器

这项技能提供了一种根据 Every 的综合风格指南审查副本的系统方法。它将克劳德变成了一位细致的行编辑和校对员，专门负责语法、结构和风格指南合规性。

## 何时使用此技能

在以下情况下使用此技能：
- 审查文章、博客文章、时事通讯或任何书面内容
- 确保文案遵循Every的特定风格约定
- 提供有关语法、标点符号和机制的反馈
- 标记与Every风格指南的偏差
- 准备干净的副本以供人工编辑审核

## 技能概述

该技能可以分四个阶段对书面内容进行全面审查：

1. **初步评估** - 了解背景和文档类型
2. **详细行编辑** - 检查每个句子是否符合要求
3. **机械审查** - 验证格式和一致性
4. **建议** - 提供可行的改进建议

## 如何使用此技能

### 第 1 步：初步评估

首先阅读整篇文章以了解：
- 文档类型（文章、知识库条目、社交帖子等）
- 目标受众
- 整体语气和声音
- 内容上下文

### 第 2 步：详细行编辑

系统地审查每个段落，检查：
- 句子结构和语法正确性
- 标点符号的使用（逗号、分号、破折号等）
- 大小写规则（尤其是职务、标题）
- 词语的选择和使用（过度使用的词语、被动语态）
- 遵守每个风格指南规则

如有疑问，请参阅完整的 [EVERY_WRITE_STYLE.md](./references/EVERY_WRITE_STYLE.md) 了解具体规则。

### 第 3 步：机械审查

验证：
- 间距和格式的一致性
- 风格选择贯穿始终
- 特殊元素（列表、引文、引文）
- 正确使用斜体和格式
- 数字格式（数字与拼写）
- 链接格式和描述

### 步骤4：输出结果

使用此结构呈现研究结果：

## 风格指南参考

完整的 Every 样式指南包含在 [EVERY_WRITE_STYLE.md](./references/EVERY_WRITE_STYLE.md) 中。需要重点关注的关键领域：

- **快速规则**：标题的标题大小写，其他地方的句子大小写
- **语气**：主动语态，避免过度使用词语（实际上，非常，只是），具体
- **数字**：拼出一到九；使用数字表示 10+
- **标点符号**：牛津逗号、不带空格的破折号、正确的引号使用
- **大写**：小写职位名称，公司为单数 (it)，团队为复数 (they)
- **强调**：仅斜体（没有粗体强调）
- **链接**：2-4 个单词，不要说“点击此处”

## 关键原则

- **具体**：始终引用带有错误的确切文本
- **参考规则**：为每次更正引用具体的样式指南规则
- **维护声音**：在纠正错误的同时保留作者的声音
- **优先考虑清晰度**：专注于提高可读性的更改
- **具有建设性**：框架反馈以帮助作家改进
- **标记不明确的情况**：当样式指南无法解决问题时，解释选项并推荐最清晰的选择

## 需要关注的共同领域

根据 Every 的风格指南，请特别注意：

- 标点符号（逗号、分号、撇号、引号）
- 大写（专有名词、标题、句子开头）
- 数字（何时拼写与使用数字）
- 被动语态（尽可能用主动语态替换）
- 过度使用的词语（实际上、非常、只是）
- 列表（平行结构、标点符号、大写）
- 连字符（复合形容词，副词除外）
- 词语使用（更少与更少，他们与他们）
- 公司参考资料（单数“it”，团队为复数“they”）
- 职称大小写
