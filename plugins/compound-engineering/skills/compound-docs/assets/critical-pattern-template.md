# 关键模式模板

将模式添加到 `docs/solutions/patterns/cora-critical-patterns.md` 时使用此模板：

---

## N. [模式名称]（始终需要）

### ❌ 错误（[将导致 X 错误]）
```[language]
[code showing wrong approach]
```


### ✅ 正确
```[language]
[code showing correct approach]
```


**为什么：** [为什么需要这样做的技术解释]

**位置/上下文：** [当适用时]

**记载于：** `docs/solutions/[category]/[filename].md`

---

**说明：**
1. 将 N 替换为下一个模式编号
2. 将[模式名称]替换为描述性标题
3. 使用导致问题的代码填写错误示例
4. 用解决方案填写正确的示例
5. 在“Why”中解释技术原因
6. 澄清此模式何时适用于“放置/上下文”
7. 链接到最初解决此问题的完整故障排除文档
