MCP 服务器参考

将 MCP 服务器视为插入 Claude Code 的强大工具。想让克劳德真正 打开浏览器 然后点击您的应用程序？那是剧作家。需要在不离开终端的情况下获得最新的 Rails 文档吗？这就是 Context7。该插件捆绑了两个服务器——它们在您安装时才起作用。

已知问题：自动加载

有时 MCP 服务器不会自动唤醒。如果 Claude 无法截取屏幕截图或查找文档，您需要手动添加它们。看 手动配置 为了修复。

剧作家

您知道如何告诉初级开发人员“打开 Chrome 并单击登录按钮”吗？现在你可以告诉克劳德同样的事情。 Playwright 让 Claude 能够控制真正的浏览器——点击按钮、填写表格、截取屏幕截图、运行 JavaScript。这就像与在你旁边打开浏览器的人结对编程一样。

提供的工具

工具 描述
browser_navigate 转到任何 URL — 您的本地主机开发服务器、生产、登台、您正在研究的竞争对手的网站
browser_take_screenshot 捕捉您现在所看到的内容。非常适合“这看起来合适吗？”设计评论
browser_click 点击按钮、链接等等。 Claude 通过文本或 CSS 选择器找到它，就像您一样
browser_fill_form 比您更快地输入表格。非常适合测试注册流程，无需手动点击
browser_snapshot 获取页面的辅助功能树 - 屏幕阅读器如何查看它。对于理解没有 HTML 噪音的结构很有用
browser_evaluate 运行页面中的任何 JavaScript。检查本地存储、触发函数、读取变量——完全控制台访问

当你使用这个时

  • 无需离开终端即可进行设计评审 - “在移动设备上截取新导航栏的屏幕截图”可在几秒钟内为您提供 PNG
  • 在编码时测试注册流程 - “使用 test@example.com 填写注册表并单击提交”为您运行测试
  • 调试生产问题 - “导航到错误页面并显示 localStorage 中的内容”无需打开 DevTools 即可显示状态
  • 竞争研究 - “前往竞争对手.com 并截取其定价页面的屏幕截图”会自动构建您的刷卡文件

用法示例

# Just talk to Claude naturally—it knows when to use Playwright

# Design review
"Take a screenshot of the login page"

# Testing a form
"Navigate to /signup and fill in the email field with test@example.com"

# Debug JavaScript state
"Go to localhost:3000 and run console.log(window.currentUser)"

# The browser runs in the background. You'll get results without switching windows.

配置

{
  "playwright": {
    "type": "stdio",
    "command": "npx",
    "args": ["-y", "@playwright/mcp@latest"],
    "env": {}
  }
}

背景7

曾经向 Claude 询问过框架并从 2023 年得到答案吗？ Context7 解决了这个问题。它是一项文档服务，可让 Claude 了解 100 多个框架的最新情况——Rails、React、Next.js、Django，无论您使用什么框架。可以将其视为将官方文档直接输入克劳德的大脑。

提供的工具

工具 描述
resolve-library-id 将“Rails”映射到 Context7 使用的实际库标识符。你不这么称呼——克劳德自动地这样做
get-library-docs 获取实际的文档页面。问“useEffect 是如何工作的？”这获取了最新的 React 文档

手动配置

如果服务器没有自动加载（你会知道，因为克劳德无法截图或获取文档），你需要自己连接它们。这是一个两分钟的复制粘贴工作。

项目级配置

要仅启用此项目，请将其添加到 .claude/settings.json 在您的项目根目录中：

{
  "mcpServers": {
    "playwright": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@playwright/mcp@latest"],
      "env": {}
    },
    "context7": {
      "type": "http",
      "url": "https://mcp.context7.com/mcp"
    }
  }
}

全局配置

或者在任何地方启用 - 您计算机上的每个项目都会获得这些服务器。添加 ~/.claude/settings.json:

{
  "mcpServers": {
    "playwright": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@playwright/mcp@latest"],
      "env": {}
    },
    "context7": {
      "type": "http",
      "url": "https://mcp.context7.com/mcp"
    }
  }
}
